Packaging Qt Project

发表于 2020-11-05

怎样打包一个Qt项目呢？

第一种方法：

QT5自带的windeployqt

首先要生成一下release版本。

这时就会找到相关文件夹。

打开这个文件夹，这时候点击exe文件是不能运行的（缺少运行所依赖的环境），将这个可执行文件复制出来，新建一个文件夹，粘贴进去。

然后打开Qt的命令行，方法是在win菜单栏Qt下找到

点击运行。进入到含有EXE文件的路径。

接下来输入windeployqt 程序名 敲击回车

然后就配置好啦！

持续更新。。。
